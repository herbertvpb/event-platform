### Principais Funcionalidade

- Scroll infinito.
- Lazy load das imagens do feed;
- Blur das imagens ainda não carregadas;
- "Pull to refresh" para atualizar o feed;

---

### Tecnologias Utilizadas

- React
- Tailwind CSS
- GraphCMS

---

### Comandos para execução

```jsx
yarn start
```
